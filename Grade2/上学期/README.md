# 大二上学期资料总览

## :book: 主要课程

大二上学期（第一学期 9 月 ～ 次年 1 月）是课程较为紧张的一个学期，主要课程有

1. （必修）大学英语（创意阅读）-3_大英
2. （必修）马克思主义基本原理_马原
3. （必修）形势与政策-3_形教
4. （必修）概率统计（理工）_概统
5. （必修）大学物理（理工）Ⅲ-2_大物
6. （必修）大学物理实验（理工）Ⅲ-2_大物实验
7. （必修）离散数学_离散
8. （必修）数据结构与算法_数据结构
9. （必修）网络安全数学基础_网安数
10. （必修）数字电子技术基础（Ⅱ）_数电
11. （必修）数字电子技术基础实验（Ⅱ）_数电实验
